## Technical Notes

### Bug 1 â€” CSS Reset

In `preview-head.html`, the reset block includes `button.topbar-create-btn`:

```css
button.sidebar-nav-item,
button.topbar-icon-btn,
button.topbar-create-btn,   /* REMOVE THIS LINE */
button.topbar-apps-btn,
button.topbar-menu-btn,
button.footer-icon-btn {
  appearance: none;
  border: none;
  background: none;   /* This overrides the purple bg! */
  ...
}
```

`button.topbar-create-btn` (specificity 0,1,1) beats `.topbar-create-btn` (0,1,0)
from the external CSS. Fix: remove it from the reset.

### Bug 2 â€” Polar Chart

Replace `PolarChartSVG` with:

- `import { PolarArea } from 'react-chartjs-2'`
- Register: `Chart.register(RadialLinearScale, ArcElement, Tooltip, Legend)`
- 6 segments matching drp- color palette (yellow, purple, mint, pink, #6f42c1, orange)

### Bug 3 â€” World Map

Replace `WorldMapSVG` with `chartjs-chart-geo` choropleth:

- Register: `ChoroplethController, GeoFeature, ColorScale, ProjectionScale`
- Fetch `countries-110m.json` from world-atlas CDN in `useEffect`
- Purple color scale matching drp- design system
