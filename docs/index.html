Overview

Bruddle Design System

A brutalist, framework-agnostic design system with offset shadows, high contrast, and 4px grid spacing.

Getting Started

Include the single CSS entry point to load all components:

<link rel="stylesheet" href="bruddle-design-system/css/index.css">

Or import selectively in your CSS:

@import 'bruddle-design-system/css/tokens.css';
@import 'bruddle-design-system/css/buttons.css';
@import 'bruddle-design-system/css/cards.css';

Quick Example

Welcome

This card uses .bru-card--raised with a button inside.

<div class="bru-card bru-card--raised">
  <div class="bru-card__header">
    <h4 class="bru-card__title">Welcome</h4>
  </div>
  <p class="bru-text">Card content here.</p>
  <button class="bru-btn bru-btn--primary">Get Started</button>
</div>

Design Principles

Brutalist Aesthetics

Bold borders, sharp edges, and unapologetic contrast. No soft gradients or rounded blobs.

Offset Shadows

Signature solid black offset shadows at multiple scales (xs through xl) give depth without blur.

High Contrast

Black on cream, white on black. Every element demands attention and passes accessibility checks.

4px Grid

All spacing derives from a 4px base unit (4, 8, 12, 16, 20, 24, 32, 40, 48, 64).

Dark Mode Built-in

Every component supports dark mode through CSS variable overrides. Add dark-mode class to body.

Framework Agnostic

Pure CSS with zero dependencies. Works with React, Vue, Svelte, vanilla HTML, or anything else.
