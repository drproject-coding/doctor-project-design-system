App Shell

App Shell Layout

The layout system provides a fixed dark sidebar, sticky topbar, scrollable content area, and optional footer. You are looking at it right now -- this docs site uses the exact same layout.

Structure

Sidebar

  • 280px wide (fixed left)
  • Dark background (#1A1A1A)
  • Collapsible to 68px
  • Brand, nav sections, user profile

Main Content

  • Fills remaining width
  • Cream background
  • Sticky topbar (68px tall)
  • Scrollable content area
  • Optional footer bar (52px)

Responsive Breakpoints

Breakpoint Max Width Behavior
Desktop large > 1200px Full sidebar (280px), 32px content padding
Desktop ≤ 1200px Reduced padding (28px)
Tablet ≤ 1024px Sidebar narrows to 260px, 24px padding
Tablet small ≤ 768px Sidebar collapses to icon-only (68px), hamburger menu appears, sidebar can overlay
Mobile ≤ 390px Sidebar hidden, mobile tab bar appears, footer hidden

Layout Variables

--bru-sidebar-width:           280px;
--bru-sidebar-collapsed-width: 68px;
--bru-topbar-height:           68px;
--bru-footer-height:           52px;
